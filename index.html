Portafolio

Algunos de los proyectos en los que trabaje

    Wipit Board

    Realizamos una plataforma para la empresa Wipit donde programadores de distintas partes del mundo puedan colaborar entre si resolviendo tareas. Desarrollada con: Javascript, React, Redux, ChakraUI, NodeJS, Express, Sequelize, SQL y PostgreSQL, Socket.io y JWT. Realizado en un equipo de 8 personas con metodología Scrum

    Arcade APP

    Web App para visualizar información sobre Videojuegos, La App permite Filtrar, Ordenar, Buscar, y Crear información, también contiene un paginado y consume datos de una API externa. Trabaje con:
    - Front: Javacript, ReactJS y Redux, CSS modules
    - Back: NodeJS, ExpressJS, Sequelize
    - Base de datos: SQL, PostgreSQ

    Movieflix

    Web App que nos permite buscar información sobre cualquier películas, agregar películas a favoritos y eliminarlas, creado con Javascript, React, Redux, HTML y css, sacando información de una Api externa.

    Weather App

    Web App que consiste en mostrar información sobre el clima en cualquier lugar del mundo, creado con Javascript, React, Redux, Html, css y jquery, la app consume datos de una api externa.

    ToDo App

    Web App para agendar las tareas que debemos realizar en el día, creada con Javascript, Html, Css y jquery.
