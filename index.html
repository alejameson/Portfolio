    Marvel App es una App creada con Javascript y React en el Front y consumiendo una Api para mostrar los Datos de los distintos personajes de Marvel. La App cuenta con un Login, un Home para seleccionar un personaje y una seccion para mostrar los Detalles del personaje

    Web App para visualizar información sobre Videojuegos, La App permite Filtrar, Ordenar, Buscar, y Crear información, también contiene un paginado y consume datos de una API externa. Trabaje con:

    App para celular que nos permite visualizar información sobre usuarios, La App permite conectar con distintos usuarios y buscar otros usuarios, ademas tenemos un carousel de distintas facultades. Es una app para universitarios, consume datos de una API externa. Trabaje con:
